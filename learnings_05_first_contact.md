# 学び 05: Granite 4.0 Nano との初対話とベンチマーク

## 状況
- セットアップした `ilab model chat` を使用し、ThinkPad (Core i5-1335U / 16GB RAM) 上でローカル推論を実行。
- モデル: `Granite-4.0-1B-Instruct-Q4_K_M.gguf`

## 結果
- **応答速度:** 初回レスポンスまで **2.388 seconds**。
- **体感:** CPUのみの推論とは思えないほど軽快。1Bクラス（Nano）を選択した戦略が、低リソース環境において極めて有効であることを証明した。
- **回答内容:** 自己紹介において「IBMとRed Hatによって開発されたGraniteモデル」であることを正しく認識。

## 考察
- 16GB RAM環境でも、モデルがメモリに完全に収まっているため、スワップによる遅延が発生していない。
- 今後の「学習（Train）」フェーズにおいても、このサイズ感であればCPUのみで実用的な時間内に完了する可能性が高い。

## 次の目標
- この「素のモデル」に対して、Taxonomy（知識体系）を追加し、特定の質問に対してより深い回答ができるよう「教育」を施す。
